# Gym Website Project

Full-stack gym management system with ASP.NET Core API and Web UI.

## Features
- User management (Admin, Trainer, Member roles)
- Gym and service management
- Appointment scheduling
- AI-powered fitness recommendations via Google Gemini
- Redis caching
- PostgreSQL database

## Quick Start (Docker)
```bash
docker compose up -d --build
```

Access the API at `http://localhost:8080`

## Development
Requires .NET 9 SDK, PostgreSQL, and Redis.

See individual project folders for more details.
